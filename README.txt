Stream Marker Mark I 

Autohotkey is required for this script to work.
https://www.autohotkey.com/


Launch with Stream_marker_mk1_launcher.ahk to comment actions with counter 
	starting at the beginning of the stream.

Log is saved to ./logs.

Mappings
{WIN + SHIFT + O} 	- create new stream log 
{ALT + O} 		- add an entry to the log
{ALT + SHIFT + O}	- add a silent entry to the log
